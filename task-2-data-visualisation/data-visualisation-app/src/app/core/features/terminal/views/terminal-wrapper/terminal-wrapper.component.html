Task 3 - Terminal Emulator to run Sanitisation Bash Script
This is an approximated terminal environment. It will allow you to run certain whitelisted commands. Any commands not already enabled by the system will fail by design. It is not possible to supply a new file via the terminal interface. Due to a quirk with shell.js trimming input characters from a flag, The only command structure that will work in this terminal is ./sanitise.sh -i "string to sanitise" . The -c flag will not be processed correctly by the shelljs.exec command.

If you are running the UI locally, make sure the node server is running by running node server.js in the node-server directory.

To use the bash script fully featured you will have to run it locally. Follow the instructions in the readme.md for Task3, which detail how to make the bash script executable and the parameters to run it.
