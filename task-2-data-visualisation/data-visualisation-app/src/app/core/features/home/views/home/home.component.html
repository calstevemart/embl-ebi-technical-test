Landing Page

Hello, this webapp serves as a way of viewing all the solutions to the EMBL-EBI technical tests that I ( Callum Martin ) have produced. The 'Machine Calculator' page corresponds to Task 1, the 'Graph' page corresponds to Task 2 and the 'Terminal' page corresponds to Task 3.

The Terminal endpoint is not a fully featured terminal. To test the bash script with custom configuration options you will have to run it locally.

If you are running this UI locally, make sure that the node server is running for the Terminal view, and that the flask app is running for the Machine calculator view.
  • To start the node server, navigate to the node-server subdirectory and run npm install && node server.js.
  • To start the flask server, you will have to first create a virtual environment and install the project. Instructions on this can be found in the Readme.md. Once you have installed the project and your venv is active, run python app.py
